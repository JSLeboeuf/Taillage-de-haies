### Mix revenue par service


### Cles pour atteindre 1M$

1. **Vitres = game changer** (180K$ de revenue additionnel, marge 75%)
   - Acheter equipement WaterFed (~3,000$)
   - 1-2 vitres/jour entre les jobs haies = 70K$ bonus
   - Nov-Dec dedie vitres = 60K$ de plus
   - Pricing: 10$/fenetre, minimum 150$/maison

2. **Gouttieres en automne** (120K$ quand les haies ralentissent)
   - Equipement minimal (souffleuse + echelle = deja la)
   - 4-5 gouttieres/jour a 200$ = 800-1,000$/jour
   - Cross-sell aux clients haies existants

3. **3e crew ete** (3 etudiants, juin-aout)
   - Cout: 42K$ (3 x 14K$)
   - Revenue additionnel: ~130K$
   - ROI: 3x en 3 mois

4. **Deneigement hivernal** (80K$ de revenue morte-saison)
   - Contrats residentiels 850-1,200$/saison
   - 50-70 contrats = 50-80K$
   - Employes PRIIME travaillent a l'annee (justifie le "non-saisonnier" du formulaire)

### Pourquoi la marge est 79%

1. **PRIIME subventionne 60%** des 2 employes = cout effectif ~8$/h au lieu de 21$/h
2. **Henri ne se paie pas** un salaire fixe (dividendes fin d'annee)
3. **3 services** sur les memes clients = revenue/client augmente sans CAC additionnel
4. **Etudiants** a 18$/h pour 3 mois seulement (pas 12 mois)

## Checklist decisions courantes

**On prend le job?**
- [ ] Ticket >= 400$? Si non → augmenter prix ou refuser
- [ ] Dans le territoire? Si >30km → ajouter 50-100$ deplacement
- [ ] Haie accessible? Si echelle >12pi → surcharge +25%

**On embauche?**
- [ ] Eligible PRIIME? Si oui → TOUJOURS (cout effectif 8$/h)
- [ ] Si non PRIIME → besoin >= 960 clients/saison pour justifier un crew TET

**On investit en marketing?**
- [ ] CAC < 40$? Si oui → augmenter budget
- [ ] CAC > 60$? Si oui → couper ce canal
- [ ] Referral programme actif? → meilleur ROI (CAC 15$)

**On achete une compagnie?**
- [ ] Prix < 250$/client? Si oui → go
- [ ] Retention historique > 70%? Si non → negocier prix
- [ ] Down payment < 30%? → ideal, structurer seller financing
