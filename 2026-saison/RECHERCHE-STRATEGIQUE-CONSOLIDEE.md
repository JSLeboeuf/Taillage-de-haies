- Lookalike 1% sur les clients convertis

- Conversion Leads API (envoyer les conversions a Meta pour optimisation)

**Phase 3 — Septembre+ (multi-services)**: ajouter vitres + gouttieres
- Meme audience, creative differente par service
- "On fait aussi vos vitres et gouttieres" sur les clients haies

