## Triple Arbitrage

1. **Capital humain** : TET mexicains loyaux (plafond 20% pour paysagisme)
2. **Capital financier** : Acquisitions en balance de vente (15% down)
3. **Capital technologique** : IA/automation pour +15-30% marge
