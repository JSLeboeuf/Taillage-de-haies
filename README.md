## Vision strategique

### Triple Arbitrage
1. **Capital humain** : TET mexicains loyaux (plafond rehausse a 20% pour paysagisme)
2. **Capital financier** : Acquisitions en balance de vente
3. **Capital technologique** : IA/automation pour +15-30% marge

### Roadmap
| Phase | CA cible | Strategie |
|-------|----------|-----------|
| An 1  | 1M$      | Valider modele OpCo/TET |
| An 2-3| 3M$      | Scaling + premieres acquisitions |
| An 5  | 10M$     | Roll-up regional |
| An 7+ | 100M$    | Plateforme PE-backed |
