# The Dream Manager - Matthew Kelly
## Aider Tes Employés à Réaliser Leurs Rêves

---

## Le Concept Central

> Les employés qui progressent vers leurs rêves personnels sont les meilleurs employés.

### Le Paradoxe
Investir dans les rêves PERSONNELS des employés → Meilleure performance PROFESSIONNELLE

---

## Pourquoi Ça Marche

| Approche Traditionnelle | Dream Manager |
|------------------------|---------------|
| "Travaille dur, tu seras récompensé" | "Réalise tes rêves, on t'aide" |
| Employé = ressource | Employé = personne |
| Loyauté par le salaire | Loyauté par l'investissement |
| Turnover élevé | Rétention exceptionnelle |

---

## Les 12 Catégories de Rêves

| Catégorie | Exemples |
|-----------|----------|
| **Physique** | Perdre du poids, marathon |
| **Émotionnel** | Paix intérieure, confiance |
| **Intellectuel** | Apprendre une langue, diplôme |
| **Spirituel** | Méditation, voyage spirituel |
| **Psychologique** | Surmonter une peur |
| **Matériel** | Maison, voiture |
| **Professionnel** | Promotion, entreprise |
| **Financier** | Économies, retraite |
| **Créatif** | Écrire un livre, musique |
| **Aventure** | Voyage, parachute |
| **Héritage** | Impact sur les autres |
| **Caractère** | Devenir meilleure personne |

---

## Le Processus

### Étape 1: Identifier les Rêves
Demander à chaque employé:
- "C'est quoi tes 100 rêves?"
- (La plupart n'en ont jamais fait la liste)

### Étape 2: Prioriser
Choisir 1-3 rêves à poursuivre activement

### Étape 3: Planifier
Créer un plan concret avec étapes

### Étape 4: Supporter
L'entreprise aide (temps, ressources, encouragement)

### Étape 5: Célébrer
Célébrer les rêves réalisés publiquement

---

## Application PME Taillage

### Pour une Équipe de 5-10 Personnes

**Investissement Temps:** 30 min/employé/mois
**Investissement $:** Variable (souvent $0)

### Exemples Concrets

| Employé | Rêve | Comment Aider |
|---------|------|---------------|
| Marc | Permis conduite | Flexibilité horaire pour cours |
| Julie | Cours comptabilité | Payer 50% du cours |
| Pierre | Acheter maison | Référence pour prêt + stabilité emploi |
| Sarah | Courir marathon | Temps off pour entraînement |

## Les 5 Niveaux d'Engagement

| Niveau | Caractéristique | % Typique |
|--------|-----------------|-----------|
| 1. Disengaged | Fait le minimum | 20% |
| 2. Engaged | Fait son travail | 50% |
| 3. Invested | Va au-delà | 20% |
| 4. Committed | Donne tout | 8% |
| 5. Missionary | Vit l'entreprise | 2% |

### Objectif Dream Manager
Déplacer les gens vers les niveaux supérieurs.

---

## ROI du Dream Manager

### Résultats Documentés (Jancoa - compagnie de nettoyage)

| Métrique | Avant | Après |
|----------|-------|-------|
| Turnover | 300%/an | 45%/an |
| Coût recrutement | Massif | Réduit 80% |
| Productivité | Baseline | +20% |
| Satisfaction client | Bonne | Excellente |

### Calcul Simple

```
Coût remplacement employé: ~$5,000 (recrutement + formation)
5 employés × 50% turnover = 2.5 départs/an = $12,500

Réduire turnover de 50% → Économie: $6,250/an
+ Meilleure performance
+ Meilleure satisfaction client
```

## Questions à Poser

### Première Conversation
1. "Si l'argent et le temps n'étaient pas un problème, qu'est-ce que tu ferais?"
2. "C'est quoi ton plus grand rêve non réalisé?"
3. "Qu'est-ce que tu voulais être quand tu étais petit?"
4. "Dans 5 ans, où te vois-tu?"

### Suivi Mensuel
1. "Comment avance ton rêve?"
2. "Qu'est-ce qui te bloque?"
3. "Comment je peux t'aider?"

---

## Intégration Simple

### Sans Programme Formel

1. **Onboarding:** Demander les rêves
2. **1-on-1 mensuels:** Suivre les rêves
3. **Opportunités:** Offrir quand possible
4. **Célébration:** Fêter les accomplissements

### Budget Minimal
- Temps: Le plus précieux
- Flexibilité: Souvent gratuite
- Formation: Partager les coûts
- Réseau: Tes contacts

## Objections et Réponses

| Objection | Réponse |
|-----------|---------|
| "Pas le temps" | 30 min/mois = 6h/an par personne |
| "Trop cher" | Coût turnover bien plus élevé |
| "Ils vont partir" | Ils partent moins, pas plus |
| "C'est leur vie privée" | C'est optionnel et volontaire |
| "Je ne suis pas thérapeute" | Tu es facilitateur, pas expert |
