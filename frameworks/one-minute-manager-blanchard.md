# The One Minute Manager - Blanchard & Johnson
## Gestion Simple et Efficace

---

## Les 3 Secrets

| Secret | Temps | Impact |
|--------|-------|--------|
| One Minute Goals | 1 min | Clarté totale |
| One Minute Praising | 1 min | Motivation boost |
| One Minute Redirect | 1 min | Correction rapide |

---

## Secret #1: One Minute Goals

### Principe
Les gens performent mieux quand ils savent EXACTEMENT ce qu'on attend d'eux.

### Règles
1. Maximum 3-5 objectifs
2. Chaque objectif sur une feuille
3. Lisible en 1 minute
4. Standard de performance clair
5. Signé par les deux parties

### Template

```
┌─────────────────────────────────────┐
│ OBJECTIF: _____________________     │
│                                     │
│ CE QUE ÇA VEUT DIRE:               │
│ _________________________________  │
│ _________________________________  │
│                                     │
│ MESURE DE SUCCÈS:                  │
│ _________________________________  │
│                                     │
│ Date: _____ Signatures: ___/___    │
└─────────────────────────────────────┘
```

### Exemple Taillage

```
OBJECTIF: Satisfaction client 100%

CE QUE ÇA VEUT DIRE:
- Haie taillée selon standard (photo)
- Débris ramassés
- Client informé à la fin

MESURE: Zéro rappel pour correction
```

## Secret #2: One Minute Praising

### Quand
Immédiatement quand tu vois quelqu'un faire quelque chose de bien.

### Comment (30 secondes)
1. **Dis spécifiquement** ce qu'ils ont bien fait
2. **Dis comment** ça te fait sentir
3. **Pause** - laisse absorber

### Puis (30 secondes)
4. **Encourage** à continuer
5. **Poignée de main** ou geste positif

### Exemple

> "Marc, j'ai vu comment tu as géré le client difficile ce matin. Tu es resté calme et tu as trouvé une solution. Ça m'impressionne vraiment. [Pause] Continue comme ça, c'est exactement ce dont on a besoin."

### Erreur Commune
Attendre la revue annuelle pour féliciter = trop tard

## Secret #3: One Minute Redirect

### Quand
Dès que tu vois une erreur ou un comportement à corriger.

### Comment (30 secondes)
1. **Confirme les faits** - ce qui s'est passé
2. **Exprime** l'impact négatif
3. **Silence** - laisse absorber

### Puis (30 secondes)
4. **Rappelle** leur valeur
5. **Réaffirme** ta confiance en eux
6. **Fini** - pas de rancune

### Exemple

> "Marc, le client a appelé - les débris étaient encore sur son terrain. Ça nous coûte du temps et de la réputation. [Pause] Tu sais que je te fais confiance, et je sais que tu peux faire mieux que ça. On n'en parle plus."

### Règles Critiques
- Jamais d'attaque personnelle
- Toujours terminer positif
- Une fois dit, c'est fini
- Privé, pas devant les autres

## Application Post-Acquisition

### Semaine 1
1. Rencontrer chaque employé
2. Clarifier 3-5 objectifs chacun
3. Écrire les objectifs ensemble

### Quotidien
1. Observer le travail
2. Praising immédiat (chercher le positif)
3. Redirect immédiat (pas d'accumulation)

### Mensuel
1. Revoir les objectifs
2. Ajuster si nécessaire
3. Feedback bidirectionnel

---

## Pourquoi Ça Marche

| Approche Traditionnelle | One Minute Manager |
|------------------------|-------------------|
| Objectifs flous | Objectifs cristal clair |
| Feedback annuel | Feedback instantané |
| Focus sur le négatif | Focus sur le positif |
| Critique = attaque | Redirect = amélioration |
| Employés incertains | Employés confiants |

---

## Ratios Importants

### Praising vs Redirect
**4:1 minimum** - 4 praisings pour 1 redirect

### Temps Manager
**80% observer, 20% intervenir**

### Fréquence
**Plusieurs fois par jour** - pas une fois par mois
