## 1.3 Profil du Vendeur Ideal

### Caracteristiques Recherchees:

| Critere | Ideal | Acceptable | Red Flag |
|---------|-------|------------|----------|
| **Age** | 60-70 ans | 55-75 ans | <50 ans |
| **Motivation** | Retraite | Sante/Fatigue | Problemes financiers |
| **Implication** | Pret a rester 6 mois | Disponible pour transition | Veut partir immediatement |
| **Financement** | Ouvert au seller financing | Negocie | Cash only |
| **Heritiers** | Aucun interesse | Pas impliques | Conflits familiaux |

### Questions de Qualification (Premier Appel):

```
1. "Depuis combien de temps avez-vous l'entreprise?"
   → Cherche: 10+ ans (Lindy Effect)

2. "Qu'est-ce qui vous amene a considerer une vente?"
   → Cherche: Retraite, fatigue, pas de releve
   → Red flag: Problemes financiers, litiges

3. "Avez-vous des enfants interesses a reprendre?"
   → Cherche: Non, ils ont d'autres carrieres
   → Red flag: Conflit familial en cours

4. "Quel serait votre timeline ideal pour une transition?"
   → Cherche: 6-12 mois, flexible
   → Red flag: "Le plus vite possible"

5. "Seriez-vous ouvert a rester quelques mois pour la transition?"
   → Cherche: Oui, fier de son legacy
   → Red flag: "Non, je veux decrocher"
```

## 1.4 Red Flags a Surveiller

### Financiers:
- [ ] Revenus en baisse 3 ans consecutifs
- [ ] Plus de 30% des revenus d'un seul client
- [ ] Aucune comptabilite formelle
- [ ] Beaucoup de cash non declare ("au noir")
- [ ] Dettes fiscales impayees

### Operationnels:
- [ ] Equipement vieux (15+ ans) sans entretien
- [ ] Aucun employe - proprio fait tout seul
- [ ] Pas de contrats ecrits avec les clients
- [ ] Pas d'assurances adequates
- [ ] Licences/permis non a jour

### Relationnels:
- [ ] Vendeur presse ou evasif
- [ ] Refuse de montrer les declarations fiscales
- [ ] Mauvaises reviews Google (< 3.5 etoiles)
- [ ] Litiges en cours ou passes
- [ ] Turnover eleve des employes

## 5.2 Les 7 Tactiques Appliquees

### TACTIQUE 1: Augmenter les Prix (+20-25%)

```
AVANT:
Prix moyen: $800/an

ACTION:
- Audit de tous les prix
- Augmenter de 20-25% les clients sous-factures
- Creer 3 tiers de service (Bronze, Argent, Or)

APRES:
Prix moyen: $1,000/an
Churn: 5% (10 clients perdus)

Revenus: $190,000 (+$30K)
```

```

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```
