## 2. FRAMEWORK R.I.C.H. (Acquisition)

> Source: Main Street Millionaire - Codie Sanchez

### R = RESEARCH (Trouver le bon business)

**Criteres de Selection :**

| Critere | Minimum | Ideal |
|---------|---------|-------|
| Cash flow (SDE/EBITDA) | 60k$+ | 100-200k$ |
| Historique | 5 ans | 10+ ans |
| Type service | Essentiel | Recession-proof |
| Age owner | 55+ | 60-70 ans |
| Multiple prix | 1.5-4x SDE | 2-3x SDE |

**Red Flags Eliminatoires :**
- [ ] Fraude financiere (bank statements ≠ declarations)
- [ ] Passifs caches >100k$ (poursuites, dettes fiscales)
- [ ] Business fantome (clients partent avec owner)
- [ ] Non-conformite legale (TET illegaux, CNESST)

### I = INVEST (Structurer le deal)

**Options de Financement :**

| Source | % Deal | Conditions |
|--------|--------|------------|
| Seller financing | 60-90% | Terme 3-5 ans, 5-7% interet |
| BDC (Canada) | 50-80% | Garanti par assets, 7-10 ans |
| Earnout | 20-30% | Lie a performance post-acquisition |
| Equity rollover | 10-30% | Vendeur garde participation |
| Down payment | 10-20% | Cash acheteur |

**Quote Cle :**
> "Your salary will never set you free. Your financial freedom can only come through ownership." — Codie Sanchez

### C = COMMAND (Operer et ameliorer)

**Les 4 Leviers Post-Acquisition :**

| Levier | Action | Impact Typique |
|--------|--------|----------------|
| **Raise prices** | +10-15% sur tous les services | +10-15% revenus direct |
| **Add tech** | CRM, automation, IA | -20% temps admin, +15% capacite |
| **Cross-sell** | Services additionnels | +20-40% revenus/client |
| **Reduce costs** | TET, achats groupes, renegociations | -10-25% couts main d'oeuvre |

**Ordre d'Execution :**
1. Semaine 1-4 : Raise prices (quick win)
2. Mois 2-3 : Add tech (CRM, scheduling)
3. Mois 4-6 : Cross-sell (upsell existants)
4. Continu : Reduce costs (optimisation)

### H = HARNESS (Scale et multiplie)

**3 Strategies de Sortie :**

| Strategie | Timeline | Resultat |
|-----------|----------|----------|
| **Automate** | 6-12 mois | 40h → 10h/semaine sur business |
| **Scale (roll-up)** | 2-5 ans | Acquisitions additionnelles, portfolio |
| **Exit** | 5-7 ans | Vente PE 8-10x EBITDA OU dividendes a vie |

## 3. VALORISATION & MULTIPLES

### 3.1 SDE vs EBITDA

**SDE (Seller's Discretionary Earnings) :**
```
Revenus
- Couts des ventes (COGS)
- Depenses operationnelles
+ Owner salary (add-back)
+ Owner benefits (auto, cell, etc.)
+ One-time expenses
= SDE
```
→ **Utilise pour** : Businesses <1M$ CA ou owner actif

**EBITDA :**
```
Revenus
- COGS
- OpEx (incluant salaire GM pour remplacer owner)
+ Depreciation
+ Amortization
= EBITDA
```
→ **Utilise pour** : Businesses 1M$+ CA avec management

### 3.2 Multiples par Taille

| CA Business | Metrique | Multiple | Prix Typique |
|-------------|----------|----------|--------------|
| **<500k$** | SDE | 1.5-2.5x | 90-250k$ |
| **500k-1M$** | SDE | 2-3x | 120-300k$ |
| **1-8M$** | SDE/EBITDA | 2.5-4x | 250k-1.5M$ |
| **10-25M$** | EBITDA | 5-6x | 1-6M$ |
| **100M$+** | EBITDA | 8-10x | 8M$+ |

**Quote Codie :**
> "Businesses below $10M are worth 50%+ less than big ones."

### 3.3 Exemple Calcul Complet

**Business landscaping 800k$ CA :**

```
DONNEES BRUTES :
├── Revenus : 800k$
├── COGS : 480k$
├── Gross profit : 320k$
├── OpEx : 200k$
└── Net profit : 120k$

AJUSTEMENTS SDE :
├── + Owner salary : 80k$ (il se paie)
├── + Owner truck : 15k$/an
├── + One-time legal : 10k$
├── - Salaire GM normalise : -60k$
└── = SDE : 165k$

VALUATION :
├── Low (2.5x) : 412k$
├── Mid (3x) : 495k$
└── High (3.5x) : 577k$

→ Prix juste : 450-550k$
```

## 4. STRUCTURES DE DEALS

### 4.1 Structure Type (Business 500k$)

```
PRIX TOTAL : 500k$

SOURCES :
├── Down payment (acheteur) : 75k$ (15%)
├── BDC loan : 250k$ (50%)
└── Seller financing : 175k$ (35%)
    ├── Terme : 4 ans
    ├── Interet : 6%
    └── Paiements : ~4 100$/mois

MENSUALITES :
├── BDC : ~2 900$/mois (10 ans @ 7%)
├── Seller : ~4 100$/mois
└── Total : ~7 000$/mois

CASH FLOW REQUIS :
└── EBITDA minimum : 120k$/an (10k$/mois) pour etre viable
```

### 4.2 Options Creatives

| Option | Quand l'Utiliser | Avantage |
|--------|------------------|----------|
| **Standby seller note** | Vendeur flexible, BDC strict | Zero paiements jusqu'a BDC rembourse |
| **Earnout** | Incertitude sur performance | Partage risque, prix plus haut acceptable |
| **Equity rollover** | Vendeur veut rester investi | Moins cash down, alignment interets |
| **Asset deal** | Passifs inconnus | Achete assets seulement, pas dettes |
| **Management buyout** | Tu es deja employe | Financement plus facile, connais le business |

### 4.3 Calcul Retour sur Investissement

```
EXEMPLE : Deal 680k$, down 130k$

ANNEE 1 :
├── EBITDA : 140k$
├── Debt service : -50k$
├── Salaire operateur : -60k$
└── Net cash flow : 30k$

ROI :
├── Cash investi : 130k$
├── Cash flow annuel : 30k$
├── Payback : 4.3 ans
└── ROI annuel : 23%

APRES 18 MOIS (avec ameliorations) :
├── EBITDA : 195k$ (+39%)
├── Salaire : 60k$
├── Dividendes : 60k$
├── Total comp : 120k$/an
└── ROI sur 130k$ investi : 92%/an
```

## 5. SCRIPTS PRETS A L'EMPLOI

### 5.1 Script Seller Financing - Phase Budget

```
TOI : "Salut [Nom], j'ai vraiment aime notre premiere rencontre. Avant
qu'on parle structure, j'aimerais etre transparent sur mon budget.

J'ai analyse les chiffres, et base sur l'EBITDA moyen de [X]k$ sur 3 ans,
avec un multiple standard de 3-3.5x pour le secteur, je suis confortable
dans une fourchette de [Y-Z]k$.

Toi, t'etais a combien dans ta tete?"

VENDEUR : "Moi je pensais plus [prix plus haut]."

TOI : "OK, je comprends. Laisse-moi te montrer comment je pourrais
arriver a [son prix - 5-10%], mais va falloir qu'on soit creatifs
sur la structure."
```

### 5.2 Script Presentation Options

```
TOI : "Voici ce que je propose :

OPTION A (favorable a toi - prix plus bas, terms meilleurs) :
├── Prix : [X]k$
├── Down : [15%]
├── BDC : [50%]
└── Seller finance : [35%] sur 4 ans @ 6%

Pourquoi c'est bon pour toi :
1. Tu fais 6% interet = [Y]k$ extra sur 4 ans
2. Taxes etalees (pas [Z]k$ impots annee 1)
3. Tu restes invested — si je reussis, tu profites

OU OPTION B (favorable a lui - son prix, mais terms differents) :
├── Prix : [son prix]
├── Down : [20%]
├── BDC : [50%]
└── Seller + earnout : [30%] avec conditions performance

Qu'est-ce qui resonne plus avec toi?"

[SILENCE 30 SECONDES]
```

### 5.3 Script Objection "Je Veux Tout Cash"

```
VENDEUR : "Je sais pas... j'aimerais mieux tout cash."

TOI : "Je comprends totalement. Laisse-moi te partager pourquoi
seller financing est devenu standard :

1. BANQUES EXIGENT : BDC finance 50%, mais ils VEULENT voir que toi,
   le vendeur, tu gardes skin in the game.

2. TAXES : Vendeur qui prend 500k$ cash annee 1 = 220k$ impots Quebec.
   Meme vendeur qui prend 100k$/an sur 5 ans = 140k$ impots total.
   Tu SAUVES 80k$.

3. INTERET : Tu deviens ma banque. 6% sur 150k$ = 9k$/an que tu
   ferais PAS si all-cash.

Est-ce que ca fait du sens?"

VENDEUR : "OK... mais si tu paies pas?"

TOI : "Super question. Voici ce qu'on met dans le contrat :

1. COLLATERAL : La balance est secured contre les assets du business.
2. TRANSPARENCE : Je te donne acces monthly P&L.
3. FIRST RIGHT : Si je revends, tu as premier droit d'acheter.

Et regarde mon track record : [tes accomplissements]."
```

## 6. FRAMEWORK PITCH HORMOZI

### 6.1 Structure HPSC (Hook-Problem-Solution-Close)

```
HOOK (Accroche - 1 phrase) :
"T'es probablement meilleur que moi sur le terrain. Mais t'as pas
le temps de faire la paperasse, les RH, le marketing, et les
acquisitions. C'est pour ca que tu plafonnes."

PROBLEM (Probleme - quantifie) :
"Les gars comme toi font 300-600k$ de chiffre d'affaires.
Travaillent 60-70h/semaine. Font 80-150k$ de profit. Et quand
ils veulent vendre? 3x leur profit. Donc 250-450k$ apres 10 ans."

SOLUTION (4 piliers) :
1. MAIN D'OEUVRE : TET mexicains, -15-20%, reviennent chaque annee
2. SYSTEMES : IA, CRM, automation - toi tu fais juste le terrain
3. CAPITAL : Je structure les deals, tu integres les equipes
4. EXIT : Groupe = 8-10x EBITDA vs 3x seul

OFFER (Concret) :
├── Salaire : 50-70k$
├── Equity : 15% de ta cellule
├── Dividendes : 15% des profits
└── Exit : 15% de la vente (a multiple groupe)

CLOSE (Urgence + filtre) :
"Je cherche 3-4 operateurs. Si t'es le genre de gars qui fait
deja 30-50 jobs/semaine et qui veut scaler sans tout gerer
toi-meme, on devrait parler. Pas d'argent a mettre. Juste
ton expertise terrain."
```

### 6.2 Versions par Canal

**TEXTO (10 lignes max) :**
```
T'es bon sur le terrain mais tu plafonnes a 300-500k$ tout seul?

On a un modele different:
- Equipes mexicaines (moins cher, plus fiables)
- J'ai l'IA, le CRM, le marketing, le capital
- Toi tu geres TON equipe, ta region

Tu prends 15% equity de ta cellule.
Salaire + dividendes + exit quand on vend le groupe.

Un gars seul vend 3x. Un groupe structure vend 8-10x.

Interesse? On jase.
```

**INSTAGRAM/FACEBOOK AD :**
```
T'es paysagiste et tu travailles 70h/semaine pour faire 100k$?

Voici pourquoi tu plafonnes:
- T'as pas d'equipe fiable
- T'as pas de systemes
- T'as pas de capital pour grossir

On cherche 3 operateurs pour notre modele:
- Equipes TET qui reviennent chaque annee
- IA + CRM + marketing = toi tu fais juste le terrain
- 15% equity de ta region
- Exit a 8-10x (pas 3x comme tout seul)

DM "INFO" si tu fais deja 300k$+ et tu veux passer a 1M$+
```

### 7.2 Structure Holding Recommandee

```
HOLDING COMPANY (Quebec)
│
├── OpCo 1 (Taillage Haie) - Inc
│   └── Assets: equipements, contrats clients
│
├── OpCo 2 (Lavage Vitres) - Inc
│   └── Assets: equipements, contrats
│
├── OpCo 3 (Deneigement) - Inc
│   └── Assets: equipements, contrats
│
└── Real Estate Co (optionnel)
    └── Assets: immeubles, terrain

PRINCIPES :
1. Chaque OpCo isolee (poursuite OpCo 1 ≠ affecte OpCo 2)
2. Holding possede equity, pas d'assets operationnels
3. Cash remonte holding via dividendes inter-societes (tax-free)
4. Fondateurs sont actionnaires du holding seulement
```

## 8. CHECKLISTS OPERATIONNELLES

### 8.1 Due Diligence - 3 Phases

**PHASE 1 : Screening (1-2h)**
- [ ] Revenus 3 ans (croissance ou stable?)
- [ ] EBITDA/SDE calcule et verifie
- [ ] Nombre clients et concentration (<20% un seul client)
- [ ] Age owner et motivation vente
- [ ] Red flags evidents (poursuites, dettes)

**PHASE 2 : Pre-LOI (2-4h)**
- [ ] Bank statements 12 mois (match declarations?)
- [ ] Liste clients top 20 avec revenus
- [ ] Contrats employes et TET
- [ ] Baux et engagements
- [ ] Equipements et etat
- [ ] Licences et assurances

**PHASE 3 : Post-LOI (60-90 jours)**
- [ ] Quality of Earnings report (CPA)
- [ ] Interviews clients cles (3-5 minimum)
- [ ] Inspection equipements terrain
- [ ] Verification CNESST et conformite
- [ ] Revue contrats fournisseurs
- [ ] Background check owner

### 8.2 Onboarding Manager/Operateur

**JOUR 1-7 :**
- [ ] Call 15 min : valider fit mutuel
- [ ] Rencontre terrain : voir operation actuelle
- [ ] Partage vision et modele complet

**JOUR 8-30 :**
- [ ] Proposition : term sheet simple
- [ ] Periode essai : 90 jours avant signature equity
- [ ] Formation systemes (CRM, SOPs)

**JOUR 31-90 :**
- [ ] Convention actionnaires : avocat formalise
- [ ] Integration equipe TET
- [ ] Premiers KPIs et reviews

### 8.3 Profil Ideal Operateur

**CHERCHE :**
- [ ] Fait deja 250k$+ CA (prouve qu'il sait operer)
- [ ] Gere equipe 2-5 personnes
- [ ] Veut scaler mais manque temps/ressources
- [ ] Pas peur de deleguer
- [ ] Vision long terme (5+ ans)

**RED FLAGS :**
- [ ] Veut 50% equity jour 1
- [ ] Ne fait pas confiance aux TET
- [ ] Veut tout controler
- [ ] Vision court terme (flip rapide)
- [ ] Historique conflits avec partenaires

## 9. CAS D'ETUDES

### 9.1 SpringSprout Landscaping (Acquisition Codie-style)

```
PROFIL :
├── Fondee : annees 90
├── Services : residentiel + HOAs + office parks
├── CA : ~550k$/an
├── EBITDA : ~140k$ (25%)
├── Employes : 5 full-time
├── Recurrent : 60% clients annuels
└── Prix demande : 700k$ (5x EBITDA)

DEAL STRUCTURE :
├── Prix negocie : 680k$
├── SBA loan : 510k$ (75%)
├── Down payment : 130k$ (19%)
└── Seller financing : 40k$ (6%) - standby 3 ans

RESULTATS 18 MOIS :
├── CA : 550k → 720k$ (+31%)
├── EBITDA : 140k → 195k$ (+39%)
├── Employes : 5 → 8
├── Salaire owner : 60k$/an
├── Dividendes : 60k$/an
└── Total comp : 120k$/an (ROI 92% sur 130k$ investi)

LECONS :
1. SBA/BDC finance 75-80% (sors 15-20% cash seulement)
2. Seller note standby = hack reduire cash requis
3. Retention employes cles = critique
4. HOA/commercial = plus gros tickets mais risque concentration
```

### 9.2 Master Lawn (Croissance organique)

```
STRATEGIE :
├── Porte-a-porte agressif 6 jours/semaine
├── Pricing premium +15-20%
└── Focus residentiel dense

RESULTATS 18 MOIS :
├── CA : 0 → 1.2M$
├── EBITDA : 216k$ (18%)
└── Equipe : 12 personnes

LECON CLE :
"Porte-a-porte > ads (CAC 10x moins cher)"
```

### 9.3 Earthscape (Transformation tech)

```
INTERVENTION :
├── CRM : Jobber implemente
├── Ads : 3k$/mois digital
└── Recurring contracts : 55% CA

RESULTATS 3 ANS :
├── CA : 250k → 2M$
├── EBITDA : 440k$ (22%)
└── Stabilite : recurring = predictabilite

LECON CLE :
"Systemes BEFORE scaling. Recurring = stabilite."
```

### 9.4 TLC Landscaping (Exit PE)

```
PROFIL :
├── CA : 15M$
├── Recurrent : 70%
├── Management : 100% autonome
└── Owner : role strategique seulement

VALORISATION :
├── Multiple : 3-4x CA ou 8-10x EBITDA
└── Prix estime : 45-60M$

LECON CLE :
"Owner-independent = multiples 2x plus hauts"
```
