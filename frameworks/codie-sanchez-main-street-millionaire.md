# 3. I - INVEST (INVESTIR) {#invest}

## 3.1 La Methode Profit Payback (Seller Financing)

> "60% of all small businesses change hands using seller financing."

### Comment ca Marche

```
Prix d'achat: $500,000
Profit annuel: $200,000
Profit mensuel: $16,000

Mise de fonds: $50,000 (10%)
Paiements mensuels: $4,000 sur 120 mois
Profit NET mensuel: $10,000 apres paiement
```

### Pourquoi le Vendeur Accepte

| Avantage Vendeur | Explication |
|------------------|-------------|
| **Plus d'argent** | Prix plus eleve qu'avec une banque |
| **Moins de taxes** | Paiements etales = bracket fiscal plus bas |
| **Annuite** | Cash flow regulier pour la retraite |
| **Fermeture rapide** | 7 jours vs 90+ jours avec banque |
| **Plus d'acheteurs** | Attire ceux qui ne qualifient pas pour un pret |
| **Legacy** | Peut transmettre a quelqu'un de confiance |

### Termes Typiques du Seller Financing

| Element | Fourchette Typique |
|---------|-------------------|
| Duree du pret | 3 - 10 ans |
| Mise de fonds | 10% - 50% |
| Taux d'interet | 3% - 8% |
| Conditions | Rapports financiers, benchmarks |

### Exemple Reel - Matt

```
Prix de l'entreprise: $500,000
Profit annuel: $200,000
Mise de fonds: $50,000 (10%)
Paiements: $4,000/mois x 120 mois + interet
Cash flow NET: $10,000/mois

ROI Annee 1: 240% ($120K net / $50K investi)
```

## 3.2 Pitch au Vendeur - Script

```
"Carl, il y a deux options:

OPTION A - Via la banque:
- Je vous paie $750,000
- La banque prend $60,000/an en interet
- Delai: 90+ jours
- Vous payez taxes sur montant total cette annee

OPTION B - Profit Payback:
- Je vous paie $1,150,000 (15% de plus!)
- VOUS recevez l'interet ($50,000+ a vous)
- Delai: 7 jours possible
- Taxes etalees sur 10 ans

Carl, comment vous preferez recevoir votre argent?"
```

## 3.3 Les 5 Autres Facons d'Acheter Sans Cash

### 1. Prets SBA (Small Business Administration)

**Avantages:**
- Mise de fonds: 10% (vs 20-30% banque traditionnelle)
- Termes: jusqu'a 25 ans
- Taux: variables mais competitifs

**Qualifications:**
- Credit score: 680+
- Pas de faillite dans les 3 dernieres annees
- Experience dans l'industrie ou gestion
- 10% minimum de mise de fonds

**Banques SBA Recommandees:**
- Live Oak Bank (liveoakbank.com)
- Celtic Bank
- Huntington National Bank
- Newtek Small Business Finance

### 2. Customer Acquisition pour $0

**Exemple Brittany - Gyms pendant COVID:**
1. Contacter gyms qui ferment
2. Offrir 50% des revenus pendant 6 mois pour transfert de clients
3. Faire un "farewell class" avec l'ancien proprietaire
4. Envoyer communications de transition aux clients

**Resultat:** $100,000+ en revenus acquis pour $0

### 3. Referral Fees pour Acquisition

**Exemple Restaurant:**
1. Restaurant ferme avec liste de 15,000 clients
2. Offrir code rabais 25% valide 1 an
3. Ancien proprio recoit commission sur chaque achat
4. Resultat: $23,000+ generes d'une liste qui allait disparaitre

### 4. Asset Acquisition

Acheter uniquement:
- Equipement a rabais
- Listes de clients
- Contrats en cours
- Marque/reputation

### 5. Earnout Structures

```
Prix total: $1,000,000
Paiement initial: $400,000
Earnout: $600,000 base sur performance future
```

# 5. H - HARNESS (EXPLOITER) {#harness}

## 5.1 Les 7 Tactiques de Croissance pour 10X les Profits

### Exemple: Power Washing Business

**AVANT (Annee 1):**
```
Leads: 500
Conversion: 35%
Jobs: 175
Prix moyen: $400
Revenus: $70,000
Depenses: $5,000
Profit: $65,000
```

### Tactique 1: Augmenter les Prix (+20%)

```
Prix: $400 → $480 (+20%)
Conversion: 35% → 33% (baisse legere)
Profit par job: +$80
```

### Tactique 2: Upsells

> "It's far more profitable to sell more stuff to your current customers than to find new customers."

| Service | Prix Initial | Avec Upsells |
|---------|-------------|--------------|
| Driveway seulement | $400 | $400 |
| + Roof cleaning | | +$300 |
| + Pool cleaning | | +$200 |
| + Deck restoration | | +$250 |
| **TOTAL** | $400 | $1,150 |

```
Prix moyen: $480 → $720 (+50%)
```

### Tactique 3: Recurring Revenue

**Offrir des abonnements:**
- Forfait biannuel
- Grouper les voisins le meme jour
- Subscriptions automatiques

```
Jobs/an: 175 → 263 (+50%)
```

**RESULTATS APRES 3 TACTIQUES:**
```
Revenus: $189,360
Depenses: $10,000
Profit: $179,360 (+176%)
```

### Tactique 4: Outsourcing

**Embaucher des techniciens:**
- 3 techniciens @ $40,000/an = $120,000
- 3 camions @ $35,000/an = $105,000
- Capacite: 700 jobs/an

```
Profit: $269,000 (3.1X de depart)
```

### Tactique 5: Sales Response Time

> "Businesses are 20X more likely to convert a lead if they reply within 5 minutes."

**Temps de reponse moyen PME:** 47 HEURES!

**Solution:** approachment.com
- Reponse en 60 secondes
- Humains reels, pas bots
- Multi-canal (text, WhatsApp, FB, IG)

```
Conversion: 33% → 50%
Jobs: 700 → 1,000
Profit: $485,000
```

### Tactique 6: Five-Star Reviews

**Statistiques Reviews:**
- 93% des clients lisent les reviews
- 90% n'achetent pas si <3.5 etoiles
- 58% paient plus pour entreprise bien notee
- Clients mecontents = 10X plus de reviews

**Solution:** nicejob.com
- Demande automatique post-service
- Sequence email/text de follow-up
- Appel si pas de reponse

### Tactique 7: Magic Moments (Content Marketing)

**Strategie YouTube pour Power Washing:**
1. Techniciens filment 1 job/semaine (timelapse)
2. Editeur overseas ($1,250/mois) monte 4 videos/semaine
3. 200 videos/an = 5M views = 20K subscribers

**Resultats:**
- SEO boost = +40 jobs/an
- YouTube AdSense = $5,000/an
- Brand deals = $15,000+

```

Conversion: 50%
Jobs: 1,140
```
