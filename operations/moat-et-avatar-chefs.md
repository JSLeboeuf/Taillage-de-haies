# Moat & Avatar Idéal - Chefs d'Équipe

> Pourquoi ils restent. Qui on veut.

## NOTRE MOAT (Au-delà du Multiple)

### Le problème : "Pourquoi il partirait pas?"

```
Chef après 2 ans chez nous :
- Il connaît les clients
- Il connaît les TET
- Il a le savoir-faire

RISQUE : Il part, ouvre sa propre business, nous vole les clients.

QUESTION : Qu'est-ce qui le retient?
```

## 8 MOATS QUI LE GARDENT

### 1. FLUX DE LEADS AUTOMATIQUE (Il ne vend pas)

```
CHEZ NOUS :
├── Agent IA répond 24/7
├── Google Ads génère 50+ leads/mois
├── Facebook Ads
├── Référencement clients existants
└── Lui = 0 temps sur acquisition

S'IL PART SEUL :
├── Doit faire porte-à-porte lui-même
├── Budget marketing? Expertise?
├── Site web? SEO?
└── 15-20h/semaine juste pour vendre

RÉALITÉ : La plupart des gars terrain DÉTESTENT vendre.
On leur enlève cette douleur.
```

**Valeur estimée** : 30-50 leads/mois × $50 CAC = $1,500-2,500/mois de valeur

---

### 2. TET CLÉS EN MAIN (Pas de paperasse immigration)

```
CHEZ NOUS :
├── On gère FERME, EIMT, IRCC
├── Logement arrangé
├── Transport aéroport
├── Contrats conformes
└── Lui = il les manage, c'est tout

S'IL PART SEUL :
├── EIMT = 3-6 mois, $1,000+ frais
├── Doit trouver logement
├── Risque erreur = amende ou refus
├── Doit former de zéro
└── TET loyaux restent avec NOUS (on les a recrutés)

RÉALITÉ : Les TET ont une relation avec Haie Lite, pas avec le chef.
S'il part, on garde nos TET.
```

**Valeur estimée** : $5,000-10,000/an en temps et frais évités


## AVATAR IDÉAL DU CHEF PARTENAIRE

### Profil Démographique

```
ÂGE : 28-42 ans
├── Assez jeune pour 5-7 ans d'engagement
├── Assez vieux pour maturité/responsabilité
└── Sweet spot : 32-38 ans

SITUATION FAMILIALE :
├── Idéal : Marié/conjointe, 1-2 enfants
├── Pourquoi : Stabilité, besoin de revenus prévisibles
├── Moins de risques de départ impulsif
└── Famille = ancrage

ÉDUCATION :
├── DEP, DEC ou expérience équivalente
├── Pas nécessaire : université
├── Nécessaire : intelligence pratique
└── Bonus : Formation gestion/entrepreneur

SITUATION FINANCIÈRE :
├── Pas endetté à outrance
├── Mais pas riche (sinon pourquoi nous?)
├── Idéal : veut améliorer son sort
└── A quelque chose à perdre (pas un gambler)
```

### Profil Psychologique

```
TRAITS ESSENTIELS (Must Have) :

✓ FIABILITÉ
  "Il fait ce qu'il dit qu'il va faire"
  - Historique d'emploi stable (pas 10 jobs en 5 ans)
  - Références qui confirment
  - Ponctuel aux entrevues

✓ LEADERSHIP NATUREL
  "Les gars le suivent"
  - A déjà géré du monde (même informellement)
  - Respecté par ses pairs
  - Sait donner des directives sans être un trou de cul

✓ OWNERSHIP MENTALITY
  "Il pense comme un propriétaire"
  - Se soucie du résultat, pas juste de ses heures
  - Voit les problèmes et les règle
  - Pas du genre "c'est pas ma job"

✓ COACHABLE
  "Il accepte le feedback"
  - Pas d'ego surdimensionné
  - Écoute avant de parler
  - Capable de dire "je me suis trompé"

✓ RÉSILIENT
  "Il bounce back"
  - A vécu des épreuves
  - Pas découragé par les obstacles
  - Mentalement tough
```

### Traits Bonus (Nice to Have)

```
○ Bilingue FR/ES
  Pas obligatoire (on a le traducteur IA)
  Mais énorme plus si oui

○ Expérience entrepreneuriale
  A déjà eu sa business (même petite)
  Comprend les sacrifices

○ Permis classe 3 ou 5
  Pour conduire remorque
  Sinon on peut former

○ Réseau local
  Connaît du monde dans sa zone
  Référencements naturels
```

### Red Flags (Ne Pas Embaucher)

```
🚩 DEALBREAKERS :

✗ "Je veux être mon propre boss"
  Traduction : Il va partir dès qu'il peut

✗ Job-hopper (5+ jobs en 5 ans)
  Traduction : Pas de loyauté

✗ Parle mal de tous ses anciens employeurs
  Traduction : Le problème c'est lui

✗ Veut le titre mais pas le travail
  Traduction : Cherche un upgrade de salaire sans effort

✗ Questions uniquement sur l'argent (première entrevue)
  Traduction : Mercenaire, partira pour $5k de plus

✗ Excuses constantes
  Traduction : Jamais responsable

✗ Négatif/cynique
  Traduction : Va pourrir l'équipe
```

## PROCESSUS DE SÉLECTION

### Étape 1 : Screening Téléphone (15 min)

```
QUESTIONS CLÉ :
1. "Parle-moi de ton expérience en gestion d'équipe"
2. "Pourquoi tu veux quitter ton job actuel?"
3. "Où tu te vois dans 5 ans?"
4. "C'est quoi ton plus grand échec professionnel?"

ÉCOUTER POUR :
- Ownership vs blâme
- Vision long terme vs court terme
- Honnêteté vs bullshit
```

### Étape 2 : Entrevue Terrain (2h avec Henri)

```
1. Tour du warehouse (30 min)
   - Observer sa réaction, questions posées
   - Semble-t-il à l'aise?

2. Rencontre équipe TET (30 min)
   - Comment interagit-il?
   - Les TET le respectent-ils?

3. Scénarios situations (30 min)
   - "Un client est mécontent, que fais-tu?"
   - "Un TET arrive en retard 3 fois, que fais-tu?"
   - "On te demande de faire 20% plus de jobs, comment?"

4. Discussion vision (30 min)
   - Expliquer modèle equity
   - Ses questions (écouter attentivement)
   - Fit culturel
```

### Étape 3 : Journée d'Essai (Payée)

```
- 1 journée terrain avec équipe existante
- Observer :
  ├── Comment il travaille physiquement
  ├── Comment il interagit avec TET
  ├── Comment il gère un imprévu
  └── Ce que l'équipe pense de lui (feedback)

APRÈS :
- Debrief avec Henri
- Go / No-Go decision
```

### Étape 4 : Offre & Onboarding

```
SI GO :
├── Offre écrite avec structure equity
├── Période probation 3 mois
├── Formation 2 semaines avec Henri
├── Assignation zone + équipe
└── Mentorat hebdomadaire mois 1-3
```
