## Guide de rencontre

### 1. Ouverture (2-3 min)

- Se presenter: qui on est, Taillage Elite, ce qu'on fait
- Mettre a l'aise: "C'est exploratoire, zero engagement"
- "Parle-moi de ton parcours, comment t'as parti ta business?"

### 2. Decouverte entreprise (5-7 min)

| Question                                                                      | Pourquoi                            |
| ----------------------------------------------------------------------------- | ----------------------------------- |
| Ca fait combien de temps que t'as ta business?                                | Anciennete = valeur de la clientele |
| C'est quoi tes services? Taillage, elagage, paysagement? En quel pourcentage? | Comprendre le mix                   |
| T'as combien de clients actifs par saison?                                    | Volume                              |
| C'est quoi le ticket moyen par job?                                           | Revenu/client                       |
| Tes clients sont ou geographiquement?                                         | Chevauchement de territoire         |
| Clients recurrents chaque annee ou one-shot?                                  | Retention                           |

### 3. Chiffres (5-7 min)

| Question                                                                     | Pourquoi               |
| ---------------------------------------------------------------------------- | ---------------------- |
| C'est quoi ton chiffre d'affaires annuel? (2-3 dernieres annees)             | Tendance de croissance |
| Si tout etait comptabilise (avec le cash), ca donne quoi?                    | Vrais chiffres         |
| C'est quoi ton profit apres toutes les depenses?                             | BAIIA                  |
| C'est quoi tes depenses principales? (salaires, gaz, equipement, assurances) | Structure de couts     |
| Combien est declare vs cash?                                                 | Levier de negociation  |

### 4. Operations (3-5 min)

| Question                                                     | Pourquoi                     |
| ------------------------------------------------------------ | ---------------------------- |
| Combien d'employes? Sont-ils formes/fiables?                 | Capacite + transferabilite   |
| Toi, tu travailles activement sur le terrain?                | Ajustement profit si il part |
| C'est quoi ton equipement? (vehicules, outils, remorques)    | Valeur des actifs            |
| Ta saison, ca va de quand a quand? Combien de jours/semaine? | Capacite revenue             |

### 5. Contrats et paiements (2-3 min)

| Question                                   | Pourquoi                   |
| ------------------------------------------ | -------------------------- |
| T'as des contrats signes avec tes clients? | Valeur comptable           |
| Les paiements: factures ou cash?           | Levier                     |
| T'as un comptable?                         | Processus de due diligence |

### 6. Intention de vente (3-5 min)

| Question                                                       | Pourquoi                         |
| -------------------------------------------------------------- | -------------------------------- |
| Qu'est-ce qui t'amene a vouloir vendre?                        | Motivation = urgence             |
| Tu voulais vendre quand? Cette annee, l'an prochain?           | Timeline                         |
| T'as une idee de ce que tu penses que ca vaut?                 | **Le laisser parler en premier** |
| Serais-tu ouvert a rester un certain temps pour la transition? | 3-5 ans ideal                    |
| Tu ferais quoi apres? Construction?                            | Confirmer le profil              |
