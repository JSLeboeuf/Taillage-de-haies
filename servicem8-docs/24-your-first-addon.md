# Your First Add-on

## Overview

This guide walks developers through creating their initial ServiceM8 add-on using a quick-start video or step-by-step instructions.

## Quick-Start Video

A video demonstration is available showing how to build a "Hello World" add-on from scratch.

## Create Your Add-on

Follow these initial setup steps:

1. Register for a ServiceM8 Developer account
2. Log in and complete account setup, then access the Developer menu
3. Create a new add-on named "Hello World"

## Configure Your Add-on

Next, configure the add-on with these steps:

4. Set the add-on type to "Simple Function Add-on (NodeJS)"
5. Upload the example manifest.json file and save
6. Click "Edit Function" to access the code editor and paste the Hello World example code

## Activate Your Add-on

To enable testing on your account:

7. Navigate to Settings > ServiceM8 Addons (the custom add-on appears in the Custom Addons section)
8. Select your add-on and click Connect

## Test Your Add-on

Verify functionality by:

9. Opening a Job from Dispatch Board or Job History
10. Locate the new "Hello Action" menu item
11. Click it to see results in a popup window

## Iterate

Modify code by clicking "Edit Function" in add-on settings. Changes take effect immediately upon saving, allowing quick testing cycles.
