# ServiceM8 Developer Documentation - Getting Started

## Overview
The ServiceM8 Getting Started page serves as the entry point for developers integrating with the ServiceM8 platform. It provides orientation toward available APIs and development resources.

## Available APIs

ServiceM8 offers eight distinct API categories:

- **Add-on SDK** - For building custom extensions
- **REST API** - Standard HTTP-based integration
- **Webhooks API** - Event-driven notifications
- **Messaging API** - Email and communication features
- **Document Templates API** - Dynamic document generation
- **Custom Fields API** - Extended data structure support
- **Feed API** - Activity stream integration
- **Account Provisioning API** - User and account management

## Development Assistance Options

For developers without in-house programming resources, ServiceM8 provides two pathways:

**1. Pre-built Solutions**
The platform maintains an "Add-ons directory" where users can locate existing integrations and extensions that may address specific needs without custom development.

**2. Expert Network**
ServiceM8 operates a "Developer Directory" connecting organizations with qualified professionals who can assist with custom projects and implementation challenges.
