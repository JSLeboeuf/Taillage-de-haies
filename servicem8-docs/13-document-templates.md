# Document Templates API - Overview

## Core Concept

The Document Templates API enables combining MS Word document templates with ServiceM8 objects (such as Job records) to generate customized documents. Merge fields within templates are automatically replaced with corresponding object data values.

## Primary Use Case

This system powers ServiceM8's internal document generation for quotes and invoices associated with Jobs. Importantly, "Addons can also use the Document Templates API to produce their own documents," extending this functionality beyond built-in features.
