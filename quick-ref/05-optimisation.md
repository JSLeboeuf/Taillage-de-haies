## Semaine 1-2: Quick Wins Immédiats

### 1. Augmenter les Prix (Jour 1)
| Action | Impact |
|--------|--------|
| +15-20% sur nouveaux clients | Immédiat |
| +10% sur renouvellements | 3-6 mois |
| Éliminer "prix d'ami" | Immédiat |

**Script pour hausse:**
> "Nos tarifs n'ont pas été ajustés depuis [X] ans. Avec l'augmentation des coûts, nous ajustons à [nouveau prix]. Votre service reste le même excellent service."

### 2. Upsells Immédiats (Jour 2)
| Service de Base | Upsell |
|-----------------|--------|
| Taille de haies | + Fertilisation |
| Entretien saisonnier | + Contrat annuel |
| Résidentiel | + Arbustes, plates-bandes |

### 3. Collecter les Comptes en Retard (Jour 3)
```
30 jours retard → Appel amical
60 jours retard → Lettre formelle
90+ jours retard → Agence de collection ou perte
```

## Mois 1: Systématiser

### Documenter les Processus
- [ ] Liste de tous les services offerts
- [ ] Prix de chaque service
- [ ] Temps moyen par service
- [ ] Coût par service
- [ ] Marge par service

### Identifier le Plus Profitable
| Service | Revenus | Marge | Temps |
|---------|---------|-------|-------|
| A | | | |
| B | | | |
| C | | | |

→ Focus sur haute marge, bas temps

### Éliminer le Non-Rentable
- Clients difficiles qui paient peu
- Services à faible marge
- Zones géographiques lointaines

## Mois 2: Ajouter Revenus Récurrents

### Transformer en Contrats Annuels
| Avant | Après |
|-------|-------|
| Appel par appel | Contrat 12 mois |
| Paiement après service | Paiement mensuel fixe |
| Client peut partir | Engagement |

### Script Conversion
> "On a une nouvelle option pour vous simplifier la vie: un forfait annuel à [prix/mois]. Ça inclut [X services] et vous n'avez plus à y penser. On s'occupe de tout."

### Ajouter Services Complémentaires
| Saison | Service Additionnel |
|--------|---------------------|
| Printemps | Nettoyage, fertilisation |
| Été | Arrosage, entretien |
| Automne | Ramassage feuilles |
| Hiver | Déneigement |

## Mois 3: Acquisition de Clients

### Réactiver Clients Dormants
> "Bonjour [Nom], on ne vous a pas vus depuis [X temps]. On se demandait si tout allait bien et si on pouvait vous aider avec votre terrain ce [saison]?"

### Demander Références
> "Connaissez-vous quelqu'un dans votre quartier qui pourrait bénéficier de nos services? Si vous nous référez quelqu'un, on vous offre [10% rabais / service gratuit]."

### Google My Business
- [ ] Profil optimisé
- [ ] Photos récentes
- [ ] Demander avis à chaque client satisfait
- [ ] Répondre à tous les avis

## Value Equation (Hormozi)

```
           Résultat Rêvé × Certitude de Résultat
Valeur = ─────────────────────────────────────────
           Temps/Effort × Sacrifices
```

### Application Taillage de Haies

| Élément | Comment Augmenter |
|---------|-------------------|
| **Résultat Rêvé** | "Terrain impeccable sans lever le petit doigt" |
| **Certitude** | Garantie satisfaction, photos avant/après |
| **Réduire Temps** | Contrat annuel = zéro appels |
| **Réduire Sacrifice** | Paiement automatique, scheduling flexible |
