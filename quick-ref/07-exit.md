# Quick Ref: Préparer une Sortie (Exit)
## Vendre pour un Maximum

---

## Le "Hat-Trick" de Création de Richesse

```
PROFIT #1: Ce que l'entreprise génère DÉJÀ quand tu l'achètes
     +
PROFIT #2: Ce que tu ajoutes par l'OPTIMISATION
     +
PROFIT #3: La VENTE pour multiple d'années de profits

= "A Lifetime of Wealth Every 3-5 Years"
```

---

## Multiples de Sortie par Type

| Type d'Entreprise | Multiple EBITDA |
|-------------------|-----------------|
| Dépendante du propriétaire | 1.5-2.5x |
| Systématisée, opérateur en place | 3-4x |
| Revenus récurrents forts | 4-6x |
| Roll-up (plusieurs combinées) | 6-10x |
| Vente stratégique | 8-15x |

### Exemple Concret

| Scénario | EBITDA | Multiple | Prix de Vente |
|----------|--------|----------|---------------|
| Achat initial | $80K | 2.5x | $200K |
| Après optimisation | $150K | 4x | $600K |
| Après roll-up | $400K | 6x | $2.4M |

---

## Ce qui Augmente le Multiple

| Facteur | Impact sur Multiple |
|---------|---------------------|
| Revenus récurrents (contrats) | +1-2x |
| Pas dépendant du proprio | +0.5-1x |
| Croissance constante (3+ ans) | +0.5-1x |
| Équipe de management en place | +0.5-1x |
| Systèmes documentés | +0.5x |
| Diversification clients | +0.5x |
| Bonne réputation (avis, NPS) | +0.5x |

---

## Ce qui Diminue le Multiple

| Facteur | Impact |
|---------|--------|
| Propriétaire = 100% des opérations | -1-2x |
| Un client > 30% des revenus | -0.5-1x |
| Pas de contrats récurrents | -1x |
| Revenus en déclin | -1-2x |
| Équipement en fin de vie | -0.5x |
| Litiges en cours | Deal killer |

## Stratégie Roll-Up

### Concept
Acheter plusieurs entreprises similaires et les combiner pour vendre l'ensemble à un multiple supérieur.

### Pourquoi ça Fonctionne
```
Entreprise A seule: $100K profit × 3x = $300K
Entreprise B seule: $100K profit × 3x = $300K
Entreprise C seule: $100K profit × 3x = $300K
                                Total: $900K

A + B + C combinées: $300K profit × 6x = $1.8M
                                Gain: +$900K
```

### Application Taillage de Haies

1. **Année 1**: Acquérir entreprise "plateforme" ($250K revenus)
2. **Année 2**: Acquérir 2 concurrents locaux
3. **Année 3**: Combiner opérations, éliminer redondances
4. **Année 4-5**: Vendre l'ensemble pour 5-7x vs 2-3x

## Types d'Acheteurs

| Type | Ce qu'ils cherchent | Multiple Payé |
|------|---------------------|---------------|
| **Individu** | Emploi + profits | 2-3x |
| **Concurrent** | Clients, territoire | 3-5x |
| **Private Equity** | Cash flow, scale | 4-6x |
| **Stratégique** | Synergies, expansion | 6-10x+ |

### Pour Taillage de Haies
Acheteurs probables:
- Chaînes nationales (ex: Nutrite, Weed Man)
- Groupes de services immobiliers
- Investisseurs roulant plusieurs entreprises de services
- Concurrent régional voulant s'agrandir

## Préparer l'Exit (18-24 mois avant)

### Année -2
- [ ] Retirer le propriétaire des opérations quotidiennes
- [ ] Engager un manager/opérateur
- [ ] Documenter tous les processus
- [ ] Convertir clients en contrats récurrents
- [ ] Nettoyer les finances

### Année -1
- [ ] Optimiser la profitabilité (couper gras)
- [ ] Montrer croissance constante
- [ ] Résoudre tout litige
- [ ] Mettre à jour équipement critique
- [ ] Préparer "data room" (tous les documents)

### 6 Mois Avant
- [ ] Engager courtier d'affaires ou aller direct
- [ ] Préparer mémorandum de vente
- [ ] Identifier acheteurs potentiels
- [ ] Fixer prix et structure idéale

---

## Documents pour la Vente

### Data Room - Contenu
- États financiers 3-5 ans
- Déclarations d'impôts 3 ans
- Liste clients + revenus
- Contrats clés
- Baux
- Liste employés + salaires
- Inventaire équipement
- Licences et permis
- Polices d'assurance
- Processus documentés

---

## Options de Structure de Vente

| Structure | Avantage Vendeur | Avantage Acheteur |
|-----------|------------------|-------------------|
| **100% Cash** | Liquidité totale | Aucun - paie plus |
| **Financement vendeur** | Prix plus élevé | Moins de cash upfront |
| **Earn-out** | Prix potentiel plus élevé | Risque réduit |
| **Equity roll** | Participe à la hausse | Alignement d'intérêts |

---

## Fiscalité de la Vente (Québec)

### Vente d'Actions vs Actifs
| Type | Imposition |
|------|------------|
| Vente d'actions | Gain en capital (50% imposable) |
| Vente d'actifs | Revenu ordinaire (plus élevé) |

### Exonération pour Gain en Capital (QSBC)
- Jusqu'à ~$970K (2024) exempt d'impôt
- Conditions à respecter (consulter comptable)

## Erreurs à Éviter

| Erreur | Conséquence |
|--------|-------------|
| Vendre sans préparation | Multiple réduit |
| Cacher des problèmes | Deal tombe ou recours |
| Prix irréaliste | Pas d'acheteurs |
| Pas de courtier | Moins d'acheteurs potentiels |
| Transition trop courte | Acheteur mécontent |
| Ignorer fiscalité | Surprises d'impôt |

## Formule de Prix Cible

```
Prix Cible = EBITDA Actuel × Multiple Cible

Où:
EBITDA = Profits + Ajustements propriétaire
Multiple = Basé sur caractéristiques (voir tableau)
```

### Exemple
```
EBITDA après optimisation: $150,000
Caractéristiques: Systématisée, 60% récurrent, manager
Multiple estimé: 4.5x

Prix Cible = $150,000 × 4.5 = $675,000
```
