# Quick Ref: Évaluer si c'est un Bon Deal
## Analyse Rapide d'Opportunité

---

## Formule de Valorisation Simple

```
Valeur = EBITDA × Multiple

EBITDA = Profits + Intérêts + Taxes + Dépréciation + Amortissement
       ≈ Profit Net + Salaire Propriétaire + Ajustements
```

### Multiples Typiques (PME Services)

| Situation | Multiple |
|-----------|----------|
| Entreprise dépendante du proprio | 1.5-2x |
| Entreprise standard | 2-3x |
| Systématisée + croissance | 3-4x |
| Récurrence forte + équipe | 4-5x |

---

## Calcul du "Profit Payback" (Codie)

```
Profit Payback = Prix d'Achat ÷ Profits Annuels

RÈGLE: Doit être < 3 ans
```

### Exemple Taillage de Haies

| Métrique | Valeur |
|----------|--------|
| Prix demandé | $300,000 |
| Revenus annuels | $400,000 |
| Profits déclarés | $60,000 |
| + Salaire propriétaire | $50,000 |
| + Dépenses personnelles | $20,000 |
| **= Profits réels (SDE)** | **$130,000** |
| **Profit Payback** | **2.3 ans ✓** |

## Check-list Due Diligence Express

### Financiers (Demander 3 ans)
- [ ] Déclarations d'impôts (T2/TP-600)
- [ ] États financiers (bilan, P&L)
- [ ] Relevés bancaires (12 mois)
- [ ] Liste clients + revenus par client
- [ ] Liste contrats récurrents

### Opérations
- [ ] Liste équipements + état/âge
- [ ] Liste employés + salaires + ancienneté
- [ ] Processus documentés?
- [ ] Logiciels utilisés?
- [ ] Assurances en place?

### Légal
- [ ] Structure légale (Inc, SENC?)
- [ ] Baux signés?
- [ ] Litiges en cours?
- [ ] Licences/permis valides?
- [ ] Contrats clients transférables?

## Ajustements à Faire au Profit

| Ajouter au Profit | Soustraire du Profit |
|-------------------|---------------------|
| Salaire du propriétaire | Salaire marché d'un manager |
| Dépenses personnelles (auto, cell) | Dépenses manquantes |
| Dépenses one-time | Revenus one-time |
| Loyer sous-marché (si proprio = local) | Loyer marché si sous-évalué |
| Amortissement | Capex maintenance requis |

---

## Zone de Fairness (Frasier)

```
Prix Plancher = 1× Profits TTM (12 derniers mois)
Prix Plafond  = Profits TTM × Multiple Industrie

Ton offre devrait être entre les deux.
```

### Négociation

| Si vendeur demande... | Tu réponds... |
|----------------------|---------------|
| Prix trop haut | "Basé sur les chiffres, voici ma valeur..." |
| Multiple élevé | "Les comparables montrent X multiple" |
| Prix ferme | "OK pour le prix, mais je veux les termes" |

---

## Red Flags Financiers

| Signal | Risque |
|--------|--------|
| Beaucoup de cash (pas de traces) | Sous-déclaration, difficile à prouver |
| Un client > 30% revenus | Dépendance, risque de perte |
| Profits en baisse 2+ ans | Tendance négative |
| Équipement vieux + pas de capex | Investissement caché requis |
| Comptes à recevoir vieux | Problèmes de collection |

## Calculateur Rapide

```
DONNÉES ENTRÉES:
Prix demandé: $________
Profits déclarés: $________
Salaire propriétaire: $________
Ajustements: $________

CALCULS:
Profits réels (SDE) = Profits + Salaire + Ajustements = $________
Profit Payback = Prix ÷ SDE = _____ ans
Multiple implicite = Prix ÷ SDE = _____x

DÉCISION:
□ Payback < 3 ans?
□ Multiple raisonnable?
□ Potentiel d'optimisation visible?

SI 3 OUI → Avancer | SI NON → Négocier ou passer
```

## Questions à Poser au Vendeur

1. "Pourquoi vendez-vous?" (Motivation réelle)
2. "Combien de temps seriez-vous prêt à rester pour transition?"
3. "Quel est votre plus gros client et quel % représente-t-il?"
4. "Vos employés clés savent-ils que vous vendez?"
5. "Qu'est-ce qui vous empêche de doubler l'entreprise?"
