# Quick Ref: Financer Sans Cash
## 220+ Façons de Financer - Les Principales

## Principe du $0 Out-of-Pocket

> "Pas de financement avec ZÉRO money down.
> Mais financement avec ZÉRO de TES dollars."

Le vendeur ou l'entreprise peut recevoir du cash à la clôture - mais pas de TA poche.

---

## Les 2 Sources de Financement

### 1. Inside Money (Argent de l'Intérieur)
Cash généré PAR l'entreprise elle-même

| Source | Comment ça marche |
|--------|------------------|
| **Dépôts clients** | Contrats payés d'avance |
| **Comptes à recevoir** | Factoring ou collection rapide |
| **Inventaire** | Vente ou crédit-inventaire |
| **Équipement** | Prêt garanti par équipement |
| **Immobilier** | Refinancement si proprio du local |
| **Cash flow** | Profits paient le vendeur |

### 2. Outside Money (Argent de l'Extérieur)
Cash de sources autres que toi

| Source | Comment ça marche |
|--------|------------------|
| **Financement vendeur** | Vendeur prête le prix |
| **Earn-out** | Paiements liés à performance |
| **Investisseurs** | OPM (Other People's Money) |
| **Prêteurs** | Banques, caisses, BDC |
| **SBA/BDC** | Prêts garantis gouvernement |
| **SPV** | Véhicule sans risque personnel |

## Structure Typique par Taille de Deal

### Petit Deal ($50K-$200K)
```
60% Financement vendeur (5 ans)
25% Earn-out (2 ans)
15% Dépôt clients / équipement
─────────────────────────────
100% sans ton cash
```

### Moyen Deal ($200K-$500K)
```
40% Financement vendeur
25% Prêt BDC/Banque (garanti par actifs)
20% Earn-out
15% Inside money
─────────────────────────────
100% sans ton cash
```

### Gros Deal ($500K+)
```
30% Financement vendeur
30% Prêt commercial
20% Investisseurs/SPV
20% Earn-out + inside money
─────────────────────────────
100% sans ton cash
```

## Financement Vendeur - Comment Demander

### Script
> "Je suis prêt à payer votre prix si nous pouvons structurer les paiements. En finançant une partie, vous recevez des intérêts (souvent 5-8%) et vous montrez votre confiance dans l'entreprise. C'est gagnant-gagnant."

### Termes Typiques
| Élément | Standard |
|---------|----------|
| Durée | 3-7 ans |
| Taux d'intérêt | 5-8% |
| Paiements | Mensuels |
| Ballon final | Possible |
| Garantie | L'entreprise elle-même |

### Pourquoi le Vendeur Dit Oui
- Revenus d'intérêts (meilleur que GIC)
- Vente fiscalement avantageuse (gain capital étalé)
- Prix plus élevé possible
- Montre confiance aux acheteurs

---

## Earn-out - Structure

### Définition
Portion du prix payée SEULEMENT si certains objectifs sont atteints après la vente.

### Objectifs Typiques
- Rétention de X% des clients
- Maintien de X$ de revenus
- EBITDA minimum de X$
- Renouvellement de contrats clés

### Exemple Taillage
```
Prix total: $300K
- $200K à la clôture
- $100K earn-out sur 2 ans:
  - $50K si 80%+ clients renouvelés An 1
  - $50K si revenus maintenus An 2
```

## Sources de Prêts au Québec

| Source | Pour Qui | Avantages |
|--------|----------|-----------|
| **BDC** | PME | Taux compétitifs, flexibles |
| **Desjardins** | Tous | Réseau local, relation |
| **Banques commerciales** | Entreprises établies | Gros montants |
| **Investissement Québec** | Croissance | Programmes spéciaux |
| **Futurpreneur** | Jeunes (<40 ans) | Mentorat inclus |

### Ce qu'ils demandent
- Plan d'affaires
- Financiers historiques (3 ans)
- Projections
- Mise de fonds (10-25% typique)
- Garanties personnelles (parfois évitables)

## SPV (Special Purpose Vehicle)

### Définition
Entité légale créée SPÉCIFIQUEMENT pour l'acquisition. Protège tes actifs personnels.

### Structure
```
Toi (0$ investi) ──┐
                   ├─► SPV (Inc/LLC) ──► Achète l'Entreprise
Investisseurs ────┘
```

### Avantages
- Zéro risque personnel
- Pas de garantie personnelle
- Limité aux actifs du SPV
- Investisseurs partagent le risque

## Erreurs à Éviter

| Erreur | Conséquence |
|--------|-------------|
| Garantie personnelle sur tout | Risque tes actifs |
| Prêt personnel | Intérêts non-déductibles |
| Tout en earn-out | Vendeur moins motivé |
| Ignorer le cash flow | Incapable de faire paiements |
| Pas de clause de défaut | Coincé si problèmes |
