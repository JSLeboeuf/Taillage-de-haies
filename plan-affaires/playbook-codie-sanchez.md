## 2. LE FRAMEWORK R.I.C.H. (Livre Main Street Millionaire)

### R = RESEARCH (Trouve le bon business)

**Criteres Codie** :
- Cash flow positif (EBITDA/SDE 60k$+)
- 5-10 ans d'historique
- Services essentiels (recession-proof)
- Owner veut sortir (55+ ans)
- Prix 1.5-4x EBITDA pour sub-1M$ businesses

### I = INVEST (Structure le deal intelligemment)

**Options financement** :
1. Seller financing 60-90%
2. SBA loan 90% (USA) / BDC 80% (Canada)
3. Earnout 20-30%
4. Equity rollover 10-30%

**Quote cle Codie** :
> "Your salary will never set you free. Your financial freedom can only come through ownership."

### C = COMMAND (Opere et ameliore)

**4 Leviers** :
1. **Raise prices** 10-15%
2. **Add tech** (CRM, automation)
3. **Cross-sell** services additionnels
4. **Reduce costs** (TET, achats groupes)

### H = HARNESS (Scale et multiplie)

**3 Strategies** :
1. **Automate** pour reduire ton temps de 40h → 10h/semaine
2. **Scale** via acquisitions additionnelles (roll-up)
3. **Exit** vente a PE 8-10x EBITDA OU garde et prends dividendes

## 3. SDE vs EBITDA : LA SCIENCE DU PRICING

### Difference Critique

**SDE (Seller's Discretionary Earnings)** :
```
Revenus
- Couts des ventes (COGS)
- Depenses operationnelles
+ Owner salary (add-back)
+ Owner benefits (auto, cell, etc.)
+ One-time expenses
= SDE
```

**Utilise pour** : Businesses <1M$ CA ou owner est actif.

**EBITDA** :
```
Revenus
- COGS
- OpEx (incluant salaire GM pour remplacer owner)
+ Depreciation
+ Amortization
= EBITDA
```

**Utilise pour** : Businesses 1M$+ CA avec management en place.

### Multiples Par Taille

| CA Business | Metrique | Multiple |
|-------------|----------|----------|
| **<1M$** | SDE | 1.5-3x |
| **1-8M$** | SDE ou EBITDA | 2.5-4x |
| **10-25M$** | EBITDA | 5-6x |
| **100M$+** | EBITDA | 8-10x |

**Quote Codie** :
> "Businesses below $10M are worth 50%+ less than big ones. At sub-$8M deals, we use 1-3x their SDE. That means the money the seller actually takes home."

### Exemple Calcul

**Business landscaping 800k$ CA** :

```
Revenus : 800k$
COGS : 480k$
Gross profit : 320k$
OpEx : 200k$
Net profit : 120k$

AJUSTEMENTS :
+ Owner salary : 80k$ (il se paie, mais GM couterait 60k$)
+ Owner truck : 15k$/an
+ One-time legal : 10k$
- Salaire GM normalized : -60k$

= SDE : 165k$

VALUATION :
├── Low end : 165k$ x 2.5x = 412k$
├── Mid : 165k$ x 3x = 495k$
└── High end : 165k$ x 3.5x = 577k$

Prix juste : 450-550k$
```

## 4. CAS REEL : ACQUISITION LANDSCAPING 700K$

### SpringSprout Landscaping

**Profil business** :
- Fonde annees 90, proprietaire Sonia
- Services : residentiel + HOAs + office parks
- **CA** : ~550k$/an
- **EBITDA** : ~140k$ (25%)
- **Employes** : 5 full-time
- **Recurrent** : 60% clients annuels
- **Prix demande** : 700k$ (5x EBITDA)

**Acheteuse** : Jolene Jones

**Structure Deal Finale** :

```
Prix negocie : 680k$

FINANCEMENT :
├── SBA 7(a) loan : 510k$ (75%)
├── Down payment (Jolene) : 130k$ (19%)
└── Seller financing : 40k$ (6%)
    ├── Terme : 3 ans
    ├── Standby (no payments until SBA paid)
    └── Interet : 5%

DEBT SERVICE :
├── SBA : 4 200$/mois (10 ans @ 7%)
├── Seller : 0$/mois (standby)
└── Total : 4 200$/mois

CASH FLOW :
├── EBITDA : 11 700$/mois (140k$/an)
├── Debt service : 4 200$/mois
├── Salary Jolene (operator) : 5 000$/mois
└── Net profit : 2 500$/mois = 30k$/an

ROI :
├── Investi : 130k$ cash
├── Cash flow annee 1 : 30k$
└── Payback : 4.3 ans
```

**Resultats 18 mois post-acquisition** :

```
CA : 550k → 720k$ (+31%)
EBITDA : 140k → 195k$ (+39%)
Employes : 5 → 8
Salary Jolene : 60k$/an
Dividendes : 60k$/an
Total comp Jolene : 120k$/an (vs 130k$ investi = ROI 92% annuel)
```

**Lecons** :
- SBA/BDC finance 75-80% (tu sors 15-20% cash seulement)
- Seller note standby = hack pour reduire cash requis
- Retention employes cles = critique (bonus/meetings)
- HOA/commercial contracts = plus gros tickets, mais plus de risque concentration

## 6. SCRIPT NEGOCIATION SELLER FINANCING

### Phase 1 : Etablir budget

**TOI** : "Salut [Nom], j'ai vraiment aime notre premiere rencontre. Avant qu'on parle structure, j'aimerais etre transparent sur mon budget. J'ai analyse les chiffres, et base sur l'EBITDA moyen de 120k$ sur 3 ans, avec un multiple standard de 3-3.5x pour le secteur, je suis confortable dans une fourchette de 360-420k$. Toi, t'etais a combien dans ta tete?"

**VENDEUR** : "Moi je pensais plus 500k$."

**TOI** : "OK, je comprends. Laisse-moi te montrer comment je pourrais arriver a 475k$, mais va falloir qu'on soit creatifs sur la structure."

### Phase 2 : Presenter seller financing

**TOI** : "Voici ce que je propose :

**Option A** (favorable a toi) :
```
Prix : 450k$
├── Down : 70k$ (15%)
├── BDC loan : 225k$ (50%)
└── Toi (seller finance) : 155k$ (35%)
    ├── Terme : 4 ans
    ├── Interet : 6%
    └── Paiements : 3 650$/mois

Pourquoi c'est bon pour toi :
1. Tu fais 6% interet = 37k$ extra sur 4 ans
2. Taxes etalees (tu paies pas 200k$ impots annee 1)
3. Tu restes "invested" — si je reussis, tu profites
```

**OU Option B** (favorable a lui) :
```
Prix : 500k$ (ton prix)
├── Down : 100k$ (20%)
├── BDC : 250k$ (50%)
└── Seller : 150k$ (30%)
    ├── Terme : 5 ans
    ├── Interet : 5%
    └── MAIS earnout : 50k$ paye seulement si EBITDA reste >110k$ annees 1-2

Pourquoi c'est bon pour toi :
1. Tu obtiens ton prix (500k$)
2. On partage le risque
3. Je suis motive a performer
```

Qu'est-ce qui resonne plus avec toi?"

**[SILENCE 30 SECONDES]**

### Phase 3 : Gerer objections

**VENDEUR** : "Je sais pas... j'aimerais mieux tout cash."

**TOI** :
> "Je comprends totalement. Laisse-moi te partager pourquoi seller financing est devenu standard :
>
> 1. **Banques exigent** : BDC va financer 50%, mais ils VEULENT voir que toi, le vendeur, tu gardes skin in the game.
>
> 2. **Taxes** : Vendeur qui prend 500k$ cash annee 1 = 220k$ impots Quebec. Meme vendeur qui prend 100k$/an sur 5 ans = 140k$ impots total. Tu sauves 80k$.
>
> 3. **Interet** : Tu deviens ma banque. 6% sur 150k$ = 9k$/an que tu ferais PAS si all-cash.
>
> Est-ce que ca fait du sens?"

**VENDEUR** : "OK... mais si tu paies pas?"

**TOI** :
> "Super question. Voici ce qu'on met dans le contrat :
>
> 1. **Collateral** : La balance de vente est secured contre les assets du business.
>
> 2. **Performance** : Je te donne acces monthly P&L.
>
> 3. **First right** : Si jamais je veux revendre, tu as premier droit d'acheter.
>
> Et honnêtement, regarde mon track record : [AutoScale AI, clinique veto, etc.]. Je batis des systemes solides."
