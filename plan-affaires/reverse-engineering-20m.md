### Formule de Valorisation

```
VALORISATION = EBITDA × MULTIPLE

20,000,000$ = EBITDA × MULTIPLE
```

## Étape 2: Comment générer 2.5M$ d'EBITDA?

### Calcul Inverse

```
EBITDA = Revenus × Marge EBITDA

2,500,000$ = Revenus × 15%
Revenus = 2,500,000$ ÷ 15%
Revenus = 16,666,667$ (~17M$)
```

### Validation avec Différentes Marges

| Marge EBITDA | Revenus Requis |
|--------------|----------------|
| 12% | 20.8M$ |
| 15% | 16.7M$ ← Cible |
| 18% | 13.9M$ |
| 20% | 12.5M$ |

**Cible:** 17M$ revenus à 15% marge = 2.5M$ EBITDA = 20M$ valorisation
