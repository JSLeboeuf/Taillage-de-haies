|---|---|
| Maison avec extérieur négligé ralentit la vente | Service de mise en valeur avant annonce (48h) |

## STRUCTURE DU PARTENARIAT

### Conditions standards

| Élément | Détail |
|---|---|
| Commission agent | 10% du contrat signé par son client |
| = sur Plan Tranquillité | **140$/client** |
| = sur Plan Propriété Immaculée | **240$/client** |
| = sur job one-time | 10% de la facture |
| Paiement commission | Virement dans les 7 jours après paiement client |
| Exclusivité | Non — l'agent peut travailler avec d'autres fournisseurs |
| Durée | Pas de terme fixe — révocable par les deux parties |

### Ce que Haie Lite offre à l'agent

1. **Service priorité 48h** pour les listings (pas les clients annuels normaux)
2. **Photos avant/après professionnelles** pour les annonces (format web + MLS)
3. **Rapport d'état extérieur** (ce qui a été fait, état de la propriété)
4. **Carte de visite co-brandée** : *"Propriété entretenue par Haie Lite"* (optionnel)
5. **Référence réciproque** : Haie Lite référe ses clients à l'agent si ils vendent

### Ce que l'agent fait

1. Identifier les clients vendeurs qui ont besoin de mise en valeur extérieure
2. Nous envoyer le contact (nom + téléphone + adresse)
3. Rien d'autre — on prend le relais

## AGENTS CIBLES — CRITÈRES DE SÉLECTION

Prioriser les agents qui :
- [ ] Font 10+ transactions / an dans les secteurs couverts
- [ ] Sont actifs sur Instagram/Facebook (opportunité de co-contenu)
- [ ] Ont des listings dans le segment 400k–1M$ (propriétaires soucieux de leur propriété)
- [ ] Sont indépendants ou dans petites équipes (plus flexibles)

### Sources pour trouver ces agents

- Centris.ca — filtrer par secteur + volume de ventes
- Instagram : chercher `#realteurLongueuil`, `#agentimmobilierRiveSud`
- Réseau personnel Henri/Jean-Samuel
- Chambre immobilière du Grand Montréal — registre public
