---

## Checklist Évaluation d'un Deal

### Go/No-Go Rapide
- [ ] Revenus > $150K
- [ ] Profits (SDE) > $50K
- [ ] Entreprise > 3 ans
- [ ] Propriétaire > 50 ans OU raison claire de vendre
- [ ] Pas de litiges majeurs

### Red Flags (= No-Go)
- [ ] Finances invérifiables
- [ ] Propriétaire = 100% des opérations
- [ ] Un client > 40% des revenus
- [ ] Litiges en cours
- [ ] Équipement en ruine
- [ ] Propriétaire hostile/irréaliste

---

## Checklist Due Diligence

### Financier
- [ ] États financiers 3 ans obtenus
- [ ] Déclarations impôts T2/TP-600 vérifiées
- [ ] Relevés bancaires 12 mois analysés
- [ ] Revenus vs banque correspondent
- [ ] Tendances 3 ans positives
- [ ] Comptes à recevoir vérifiés

### Opérationnel
- [ ] Liste clients + revenus obtenue
- [ ] Contrats actifs vérifiés
- [ ] Liste employés + salaires
- [ ] État équipement inspecté
- [ ] Processus quotidiens compris
- [ ] Qui reste après transition?

### Légal
- [ ] Structure corporative vérifiée
- [ ] Baux et contrats revus
- [ ] Litiges recherchés
- [ ] Licences/permis valides
- [ ] Assurances adéquates

## Checklist Calcul Valeur

```
ÉTAPE 1: SDE (Seller's Discretionary Earnings)
Profit Net déclaré:           $_______
+ Salaire propriétaire:       $_______
+ Avantages perso:            $_______
+ Dépenses one-time:          $_______
+ Amortissement:              $_______
- Dépenses manquantes:        $_______
= SDE:                        $_______

ÉTAPE 2: Multiple
Base (services):               2.5x
Récurrence 60%+:              +0.5
Manager en place:             +0.5
Croissance 3+ ans:            +0.5
Dépendant proprio:            -0.5
Un gros client:               -0.5
= Multiple Final:             ___x

ÉTAPE 3: Valeur
Valeur = SDE × Multiple = $_______

ÉTAPE 4: Vérification
Profit Payback = Prix ÷ SDE = ___ ans
✓ Doit être < 3 ans
```

# PHASE 3: POST-ACQUISITION

## Checklist Jour 1-7

### Jour 1
- [ ] Rencontrer tous les employés
- [ ] Message clair: continuité + amélioration
- [ ] Collecter tous les accès (banque, systèmes)
- [ ] Comprendre l'horaire de la semaine

### Jour 2-3
- [ ] 1-on-1 avec chaque employé clé
- [ ] Appeler 5 clients importants (avec ancien proprio)
- [ ] Comprendre les systèmes actuels
- [ ] Noter TOUS les problèmes mentionnés

### Jour 4-7
- [ ] Augmenter prix nouveaux clients +15%
- [ ] Lister tous les upsells possibles
- [ ] Collecter comptes 60+ jours
- [ ] Identifier top 20% clients

# PHASE 4: PROTECTION & EXIT

## Checklist Protection Actifs

### Immédiat
- [ ] Comptes perso/entreprise séparés
- [ ] Assurance responsabilité civile ($2M+)
- [ ] Assurance automobile commerciale
- [ ] Clauses limitatives dans contrats clients

### Court Terme
- [ ] Structure holding consultée avec comptable
- [ ] Formalités corporatives à jour
- [ ] REER/CELI maximisés

---

## Checklist Préparation Exit (18-24 mois avant)

### Année -2
- [ ] Retirer propriétaire des opérations
- [ ] Manager en place
- [ ] Processus documentés
- [ ] Clients convertis en contrats récurrents

### Année -1
- [ ] Profitabilité optimisée
- [ ] Croissance constante visible
- [ ] Litiges résolus
- [ ] Équipement mis à jour
- [ ] "Data room" préparée

### 6 Mois Avant
- [ ] Courtier engagé (ou stratégie directe)
- [ ] Mémorandum de vente prêt
- [ ] Acheteurs potentiels identifiés
- [ ] Prix et structure définis
